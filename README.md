# Stockyard Chalkboard

**Internal wiki.** Markdown pages, full-text search, version history, nested pages. Simpler than Notion, more permanent. Single binary, no external dependencies.

Part of the [Stockyard](https://stockyard.dev) suite of self-hosted developer tools.

## Free vs Pro

| Feature | Free | Pro ($4.99/mo) |
|---------|------|----------------|
| Pages | 20 | Unlimited |
| Full-text search | ✓ | ✓ |
| Public wiki | ✓ | ✓ |
| Version history | — | ✓ |
